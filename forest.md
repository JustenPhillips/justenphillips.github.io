## Synopsis
Forest taught me A LOT about Domain Controllers and forced me on a journey to understand as much as I could about Windows AD Authentication mechanisms. This box was rated as Easy but if you're not a Expert in AD attack vectors like me then I'd easily rate this as hard as it took me a few weeks to fully grasp. I was able to root this box while it was active back in October,2018. It consist of a pre-auth vulnerability and a few ACL misconfigurations that allowed us to eventually gain Domain Admin.

Now that we are on the system with this service account, let's find out exactly just how much permission we have here. There is a tool available for this.

**BloodHound**

BloodHound (https://github.com/BloodHoundAD/BloodHound) is an application used to visualize active directory environments. The front-end is built on electron and the back-end is a Neo4j database, the data leveraged is pulled from a series of data collectors also referred to as ingestors which come in PowerShell and C# flavours.

It can be used on engagements to identify different attack paths in Active Directory (AD), this encompasses access control lists (ACLs), users, groups, trust relationships and unique AD objects. The tool can be leveraged to find different paths to targets. 

when you’ve compromised an endpoint on a domain as a user you’ll want to start to map out the trust relationships, enter Sharphound for this task. It needs to be run on an endpoint to do this, as there are two flavours (technically three if we include the python ingestor) we’ll want to drop either the PowerShell version or the C# binary onto the machine to enumerate the domain.

It isn’t advised that you drop a binary on the box if you can help it as this is poor operational security, you can however load the binary into memory using reflection techniques.

The syntax for running a full collection on the network is as follows, this will use all of the collection method techniques in an attempt to enumerate as much of the network as possible.

The run Bloodhound on your attack box and it will digest the data and return every possible privledge esculation to a Domain group/user.
